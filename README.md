# Geryme Vega | Personal Portfolio 🚀

> A minimalist, high-performance static portfolio built with modern vanilla web technologies, featuring a dynamic CSS-grid architecture and ambient interaction.

## 💡 Overview

This repository contains the source code for my personal portfolio. As a systems-focused student and DOST-SEI Scholar, I wanted a website that reflects my engineering philosophy: **clean, efficient, and free of unnecessary bloat**. 

Instead of relying on heavy JavaScript frameworks, this site leverages native CSS capabilities, semantic HTML, and lightweight DOM manipulation to achieve a premium, interactive experience.

## ✨ Key Features

*   **Asymmetrical Split-Pane Layout:** A sticky identity pane paired with a scrolling content pane for seamless navigation.
*   **Ambient Cursor Interaction:** A highly optimized radial gradient that tracks mouse movement using `requestAnimationFrame` for buttery-smooth 60fps rendering.
*   **Blueprint Grid Architecture:** A purely CSS-generated background grid that scales perfectly across devices without the performance hit of canvas particles.
*   **Staggered Typography Reveal:** Custom CSS clip-path animations triggered sequentially for a polished load-in experience.
*   **Native Dark/Light Mode:** A fully functioning theme controller mapped to custom CSS variables (`--bg-base`, `--text-primary`, etc.) for instant, flicker-free switching.
*   **Intersection Observer Integration:** Content modules naturally fade and slide into the viewport only when requested by the user's scroll position.

## 🛠️ Tech Stack

*   **HTML5** (Semantic structure, SVG injection)
*   **CSS3** (Custom properties, Flexbox/Grid, Keyframe animations, Backdrop-filters)
*   **JavaScript** (Vanilla ES6, DOM Manipulation, Intersection Observer API)
*   **Zero Dependencies:** No React, no Tailwind, no jQuery. Just pure, native web technologies.
